# aintogeny

Static homepage for [aintogeny.com](https://aintogeny.com/): AI-native systems that mature with your business.

## Commands

```bash
npm run validate
npm run build
npm run check
npm start
```

- Source files live in `src/`.
- `npm run build` copies the static site into `dist/`.
- `npm start` serves `dist/` when present, otherwise `src/`, at `http://localhost:4173`.
